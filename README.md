# Audit Management Dashboard
A professional, open-source platform designed to streamline audit workflows, track compliance by department, and automate reporting.

## Key Features
- Departmental Scoping: Organize audits by organizational structure.
- Real-time Tracking: Monitor audit progress and pending tasks.
- Modular Architecture: Easy to extend for different industries (ISO, Security, Finance).
- Exportable Reports: Generate insights from audit data (Ready to implement).

# Technology Stack
| Technology | Role |
| :--- | :--- |
| **Nuxt 3** | Full-stack Framework (SSR/SPA) |
| **Vue 3** | Reactive UI Components (Composition API) |
| **Nitro** | Next-generation Server Engine |
| **TypeScript** | Type-safe development |
| **npm** | Package manager |

## Quick start
```bash
# Install dependencies
npm install

# Run development server
npm run dev```
